[ PC 에 경로설정 ]

-> 창에서 [환경변수...] 클릭!

환경변수....란? 
-> 프로그램 실행시 전역적으로 경로를 등록하여
잡아주는 곳!

"JAVA_HOME" 이름으로 jdk11 버전 폴더까지 잡아 등록!
path 에 "%JAVA_HOME%\bin" 을 등록한다!
앞의 경로가 "JAVA_HOME"에 담겨 있고 %(퍼센트)사이에 이것을 씀!
bin폴더까지 잡아주면 자바실행 파일이 전역적으로 등록되어
어디서나 호출가능!!!

[ 등록후 확인 ]
1. cmd 창 실행(실행에  cmd 엔터!)
2. java -version 확인! -> 버전이 나옴
3. javac -version확인! -> 버전이 나옴
-> 만약 확인 안될 경우 재부팅~! 환경변수 등록여부 재확인!
